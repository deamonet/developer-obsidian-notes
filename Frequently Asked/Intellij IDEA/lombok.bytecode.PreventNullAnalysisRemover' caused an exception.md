**Describe the bug**  
When running gradle clean build, I get an exception, stacktrace attached below, when transforming my class to use [@builder](https://github.com/builder) annotation.

**To Reproduce**  
gradlew clean build to this class

```

@lombok.AllArgsConstructor
@lombok.NoArgsConstructor
@lombok.Builder(toBuilder = true)
@lombok.EqualsAndHashCode(callSuper = true)
@lombok.Data
public class PlayerProfile extends BaseEntity {

  @NotNull
  @Indexed(unique = true)
  private String externalId;

  @NotNull

    super(id, lockVersion);
    this.externalId = externalId;
  }

  public PlayerProfile(String externalId) {
    this(null, null, externalId);
  }

}

Base entity isn't important to reproduce the error. The key part is @Builder.Default.
